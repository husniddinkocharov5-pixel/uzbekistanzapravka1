## O'rnatish va Ishga tushirish

1. **Repozitoriyni yuklab oling:**
   ```bash
   git clone https://github.com/your-username/uzbekistan-zapravka-info.git
   cd uzbekistan-zapravka-info
   ```

2. **Kutubxonalarni o'rnating:**
   ```bash
   npm install
   ```

3. **.env faylni yarating:**
   Loyiha ildizida `.env` faylini yarating va Google Gemini API kalitingizni kiriting:
   ```
   API_KEY=sizning_google_api_kalitingiz
   ```

4. **Ilovani ishga tushiring:**
   ```bash
   npm run dev
   ```
